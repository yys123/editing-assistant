# 01 图片识别

**文件**: `backend/services/gemini.py`
**函数**: `analyze_image()`
**调用方式**: Gemini 视觉模型，直接传入图片二进制数据

## System Prompt

无独立 System Prompt，使用默认模型配置。

## 提示词

```
这是一张医学图表，标题为 {caption_hint}。
请提取图中所有文字内容，按逻辑顺序整理输出。
如果是流程图，按步骤描述流程；如果是表格，按行列整理；如果是其他类型，提取所有关键信息。
只输出提取到的内容，不要描述图的格式或添加额外说明。
```

## 输入

| 参数 | 说明 |
|------|------|
| `image_bytes` | 图片二进制数据 |
| `mime_type` | 图片MIME类型 |
| `caption` | 图片标题/说明（可选） |

## 输出

纯文本：提取的图表文字内容。

## 使用场景

文章上传解析阶段，遇到内嵌图片时自动调用，提取流程图、表格等图表中的文字内容，作为章节 content 的一部分。
